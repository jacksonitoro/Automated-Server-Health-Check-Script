
# Automated Server Health Check Script

## Monitors mulitple Linux servers via SSH and checks:

- CPU usage
- Memory usage
- Disk space
- Running services

Sends email alerts when thresholds are exceeded.

## Tech Stack

- Python
- Paramiko (SSH)
- Linux commands
- SMTP (Email alerts)

## How to Run

1. Create virtual environment
2. Install requirements
3. Configure .env
4. Run main.py
